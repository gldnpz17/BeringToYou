Mitra Kami

Beringharjo.co.id atau Yayasan Beringharjo Inisiatif Indonesia didirikan pada tahun 2020 di pasar beringharjo, Yogyakarta. Pendirian yayasan ini awalnya ditujukan untuk memberdayakan para pedagang di Pasar Beringharjo sebagai bantuan pemasaran produk pedagang kepada pembeli secara daring melalui media sosial, agar pedagang termasuk pedagang tradisional mampu bersaing di era digital khususnya pada saat menghadapi pandemi COVID-19.
